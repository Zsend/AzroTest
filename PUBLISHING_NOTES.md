# BaseRelay Publishing Notes

This folder is the full static site package. Upload the contents of this folder to the web host root.

Before final public launch, verify:
- final domain/canonical URL;
- form destination/CRM;
- brand/name clearance;
- final owner review of compatibility, safety, warranty, and support language.

The site is intentionally validation-list only: no ecommerce, no hardware sale, no paid reservation, and no inventory commitment.

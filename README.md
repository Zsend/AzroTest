# Sweetie & Oso Puppy Website

This is a polished static website for the planned Sweetie x Oso doodle litter in Riverton, Utah.

## Included

- `index.html` - main website
- `styles.css` - modern responsive styling
- `script.js` - gallery filters, lightbox, mobile navigation
- `assets/photos/` - optimized full-size photos
- `assets/thumbs/` - optimized thumbnail photos
- `assets/photo-manifest.json` - photo metadata and categories
- `assets/photo-contact-sheet.jpg` - quick visual reference of included photos
- `assets/og-image.jpg` - social/preview image
- `assets/videos/` - placeholder folder for future video files

## Current content assumptions

- Sweetie and Oso are the planned puppy parents.
- The attached apricot/red dog photo is Oso.
- Willow and Finn are listed as Sweetie's parents.
- Oso's parent information is intentionally marked as coming soon.
- The site says the litter is in planning stage and does not claim pregnancy or availability yet.

## Before publishing

1. Replace `YOUR_EMAIL_HERE@example.com` in `index.html` with the correct contact email or connect the form to Google Forms/Formspree.
2. Add final parent details: sizes, weights, coats, temperaments, health testing, and any vet/reproduction results you want public.
3. Remove photos that feel too private, blurry, repetitive, or less professional.
4. Add puppy photos, puppy cards, and videos after the litter arrives.
5. Do not publish health claims until they are verified.

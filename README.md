# Reserve Standard — final publish-ready site

This bundle is the cleaned publish-ready static site for Reserve Standard / Reserve Standard LLC.

## Included
- `index.html` — main homepage
- `standard-brief.html` — branded reserve brief page
- `privacy-policy.html` — starter privacy page
- `risk-disclosure.html` — starter risk disclosure page
- `terms-of-service.html` — starter terms page
- `assets/` — logos, favicons, screenshots, OG card
- redirect stubs (`about.html`, `contact.html`, etc.)
- `CNAME`, `robots.txt`, and `sitemap.xml`

## Before publishing
- confirm the final domain in `CNAME`, canonical tags, and sitemap
- confirm the inbox at `hello@reservestandard.com`
- review legal pages with counsel before going live
- replace any placeholder business details if needed

## Deploy
Upload the folder contents to any static host (Cloudflare Pages, Netlify, Vercel static output, GitHub Pages, S3/CloudFront, etc.).

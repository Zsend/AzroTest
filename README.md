# CragLink Power final customer-ready static site

This package contains a responsive static website for CragLink Power.

## Files
- `index.html` — primary landing page
- `system.html` — system architecture page
- `field-team.html` — athlete / field proof page
- `styles.css` — full responsive design system
- `app.js` — mobile nav and prototype form capture
- `assets/` — logo, favicon, product render, field map

## Launch notes
- Forms are Netlify-compatible (`data-netlify="true"`) and also store submissions locally for prototype testing. Before sending paid traffic, connect the forms to Netlify, Tally, HubSpot, Shopify, Formspree, or another CRM.
- Product specs and price bands are labeled as targets. Final specs, warranty, certifications, and pricing should be confirmed after engineering and supplier quotes.
- Starlink Mini is sold separately. Do not imply affiliation with Starlink or SpaceX without a formal agreement.

## Color strategy
- Header / hero: Arctic Cyan + Midnight Navy = signal, sky, trust, satellite precision.
- Middle site: Glacier Mint + Seafoam Sage = plants, recovery, nature, outdoor calm.
- Footer / conversion close: Alpine Gold + Copper Ember = dirt, warmth, durability, premium expedition finish.

# Reserve Standard — publish-ready static site

This bundle is the polished white business version with locked desktop chapter transitions.

## Included
- Home page with section-lock desktop scroll
- Reserve Brief page
- Privacy, risk, and terms pages
- Brand assets, favicons, robots, sitemap, and manifest
- Placeholder domain and email values ready for replacement

## Desktop behavior
On desktop-sized screens, one wheel or trackpad gesture advances one full section and hard-settles on that chapter.
On smaller screens, the site falls back to normal scrolling.

## Replace before launch
- `your-domain.com`
- `hello@your-domain.com`
- rename `CNAME.example` to `CNAME` once the final domain is set

## Launch
Deploy the folder as a static site.
Open `index.html` locally for review.

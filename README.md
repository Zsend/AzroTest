# BaseRelay — Final Ready-to-Publish Site Package

This package was built from the actual uploaded `the uploaded source site package` source ZIP. It preserves the original site structure, CSS, JavaScript, background/product imagery, and logo visual language while rebranding the public copy to BaseRelay and cleaning unsafe launch/sales claims.

## What is included
- `index.html` homepage
- `field-team.html` field validation page
- `legal.html` compatibility, safety, and terms page
- `privacy.html` privacy page
- `success.html`, `404.html`
- `styles.css`, `script.js`
- `/assets` with the original visual system and BaseRelay-updated logo/product imagery
- `robots.txt`, `sitemap.xml`, `_headers`, `_redirects`

## Publishing
Upload the contents of this folder to the web host root. Do not upload only `index.html`.

## Brand note
The original logo symbol/visual language was preserved and the wordmark was updated to BaseRelay. The old source-brand logo was archived in the reference files, not used in deploy.

## Safety note
This site does not sell hardware, take payments, promise runtime, imply partner endorsement, or claim certification. It is built for validation-list publishing and field-test recruitment.

# Reserve Standard LLC site

This bundle is the publish-ready static site for the Reserve Standard business homepage.

## Key behavior
- Brand updated to **Reserve Standard / Reserve Standard LLC**
- Desktop uses a **chapter-lock scroll system**
- One wheel or trackpad gesture advances to the next section and auto-stops there
- Guided mode only turns on when every chapter fits cleanly on the current screen
- Smaller screens and reduced-motion users fall back to normal free scrolling

## Main files
- `index.html` — homepage
- `style.css` — full site styling
- `script.js` — chapter-lock scroll logic, nav state, and form behavior
- `brief.html` — Standard Brief page
- `legal/` — privacy, risk, and terms
- `assets/` — icons, screenshots, and OG card

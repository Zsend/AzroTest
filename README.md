# CragLink Power — customer-ready static site

This is a launch-ready static site package built around the approved visual direction: premium outdoor hardware, Arctic Cyan sky/signal at the top, Glacier Mint/Seafoam Sage nature layer through the middle, and Alpine Gold/Copper Ember warmth in the footer.

## Files

- `index.html` — final homepage
- `field-team.html` — athlete/field proof program page
- `legal.html` — compatibility and launch boundary page
- `privacy.html` — privacy starter page
- `styles.css` — full responsive design system
- `script.js` — mobile nav and local form preview behavior
- `assets/` — logo, favicon, mountain SVG, product visualization, OG cover

## Before publishing

1. Connect the founder form to Netlify Forms, Tally, HubSpot, Shopify, Formspree, or another CRM.
2. Replace placeholder legal/privacy language with counsel-reviewed terms.
3. Confirm final specs, safety language, warranty terms, and pricing before collecting payment.
4. Do not imply Starlink affiliation unless a formal partnership exists.
5. Replace concept product visuals with real product photography/renders when available.

## Form behavior

The founder form includes Netlify Forms attributes. When opened locally, JavaScript prevents submission and shows a preview success message. On Netlify, remove that local-only condition only if needed; the current script allows normal submission on non-local deployed domains.

# CragLink exact customer-ready site

This build is intentionally pixel-matched to the approved homepage artwork. The visual layer is the approved design image, while an invisible interaction layer makes navigation, CTA buttons, modals, forms, footer links, and mobile/tablet actions work.

## Publish
Upload the entire folder, not just `index.html`.

Required:
- `index.html`
- `styles.css`
- `script.js`
- `assets/`
- `field-team.html`
- `legal.html`
- `privacy.html`
- `success.html`

## Forms
The founder form validates and shows success in static preview. Connect it to Netlify Forms, Formspree, HubSpot, Shopify, or another CRM before sending real traffic.

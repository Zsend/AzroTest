# Reserve Standard — publish-ready site

This bundle contains the final Reserve Standard marketing site and supporting static pages.

## What is included

- Homepage with desktop full-page section lock experience
- Reserve brief page
- Privacy, risk, and terms pages
- Brand assets, favicon set, and OG card
- Redirect/support pages from the earlier site structure
- `CNAME.example`, `robots.txt`, `sitemap.xml`, and `site.webmanifest`

## Important placeholders to update before launch

- `your-domain.com`
- `hello@your-domain.com`
- Rename `CNAME.example` to `CNAME` after the final domain is chosen

## Desktop scroll behavior

On larger desktop screens, the homepage switches into a locked chapter viewport:

- one wheel or trackpad gesture advances one full section
- each section settles cleanly in place under the header
- header links and the right-side rail jump directly to each section

Mobile and smaller screens fall back to normal page scrolling.

## Contact flow

The Reserve Review form is static. It opens a pre-filled email draft instead of submitting to a backend.

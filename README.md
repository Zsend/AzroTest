# Proof Labs Publish-Ready Visual Private Site

This is a static, highly visual Proof Labs website package for private/password/no-index staging.

## Open locally

Open `index.html` in a browser.

## Pages

- `index.html`
- `one-pager.html`
- `evidence.html`
- `standard-cva.html`
- `audit.html`
- `breakglass.html`
- `quartzpath.html`
- `registry.html`
- `claim-safe.html`

## Deployment

Deploy the folder to a static host such as Netlify, Vercel, Cloudflare Pages, or GitHub Pages. Use password protection/private links where available.

## Boundary

This package is ready to deploy as a private staging site. It is not approval for public launch, public claims, cannabis sales, plant-touching operations, health/safety claims, patent claims, or technical mechanism disclosure.

# CragLink Power — publish-ready site package

This package is built to match the approved visual mockup direction: dark premium outdoor hardware, Arctic Cyan sky/signal in the hero, Glacier Mint + Seafoam Sage in the middle, Alpine Gold + Copper Ember in the footer, and the corrected CragLink logo treatment.

## Files
- `index.html` — homepage
- `field-team.html` — field credibility / athlete seeding page
- `legal.html` — starter compatibility / affiliation note
- `privacy.html` — starter privacy note
- `styles.css` — responsive visual system
- `script.js` — mobile menu + static form fallback
- `assets/` — logo, product image, background SVGs, favicon, noise texture

## Launch steps
1. Replace prelaunch/speculative copy with final engineering-approved product specs before accepting payment.
2. Connect forms to Netlify Forms, Tally, HubSpot, Shopify, Formspree, or your CRM.
3. Replace placeholder legal/privacy pages with real policies.
4. Compress production images and upload the folder to Netlify, Vercel, Cloudflare Pages, or Shopify.

## Design notes
The CTA stays cyan across the site to preserve conversion clarity. Warm colors are used in the bottom/footer only to create earth/basecamp warmth without breaking the brand's technical trust.

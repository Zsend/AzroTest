# CragLink Power — production site package

This package is a real HTML/CSS/JS rebuild of the approved visual direction. It does not use the full mockup as a pasted screenshot. Text, buttons, navigation, forms, modals, and links are real site elements.

## Publish
Upload the entire folder contents, including:

- `index.html`
- `field-team.html`
- `legal.html`
- `privacy.html`
- `success.html`
- `styles.css`
- `script.js`
- `/assets`
- `robots.txt`
- `sitemap.xml`

Do not upload only `index.html`; the site needs the CSS, JS, and image assets.

## Forms
The founder forms validate and store preview submissions in browser localStorage, then route to `success.html`. Before real traffic, connect forms to Netlify Forms, Tally, HubSpot, Shopify, Formspree, Klaviyo, or another CRM.

## Brand design system
- Header / hero: Arctic Cyan + Midnight Navy = satellite, sky, signal, precision.
- Middle: Glacier Mint + Seafoam Sage = nature, plants, field trust.
- Bottom / footer: Alpine Gold + Copper Ember = dirt, warmth, basecamp, expedition energy.
- Logo: top beacon remains clean; no random mixed-color sparkle treatment.

## Compatibility language
The copy avoids implying official Starlink affiliation. Keep that standard until a formal partnership exists.

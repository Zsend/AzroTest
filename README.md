# CragLink Exact Artboard Site

This version is built to match the approved image exactly by using the approved desktop/tablet/mobile artboards as the visual layer and invisible working hotspots for interactivity.

Why this approach: the approved design is a raster mockup, not a layered Figma file. A component rebuild cannot be pixel-perfect without rebuilding every asset and section from layers. This package prioritizes exact visual match and working buttons without visible stamped overlays.

Publish the full folder, including `assets/`, `styles.css`, and `script.js`.

Before real traffic, connect forms to a real backend such as Netlify Forms, Tally, HubSpot, Shopify, Formspree, or your CRM.

# CragLink Power final responsive site

Upload every file in this folder to your host. Do not upload `index.html` by itself; the site uses `styles.css`, `script.js`, and SVG assets in `/assets`.

## What works
- Real responsive HTML/CSS layout, not an image-only mockup.
- All buttons and navigation links work.
- Modal founder list and footer forms validate email and show a success toast.
- Forms store leads in browser localStorage for static preview only.

## Before launch
Connect forms to Netlify Forms, Tally, HubSpot, Shopify, Formspree, or another backend so leads are captured server-side.

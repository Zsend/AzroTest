# CragLink Power — pixel-matched working site

This package is locked to the approved image direction. It uses the approved desktop/tablet/mobile artwork as the visual layer and adds real clickable hotspots, modal forms, quick links, responsive behavior, and starter legal/support pages.

## Publish
Upload the folder contents to Netlify, Vercel, GitHub Pages, Squarespace custom code, or any static host.

## Forms
The forms are marked up for Netlify Forms. In local/GitHub preview, JavaScript shows a success toast. For production lead capture, connect Netlify Forms, Tally, HubSpot, Formspree, Shopify, or your CRM.

## Files
- `index.html` — homepage matching approved mockup artwork
- `field-team.html` — field team/athlete credibility page
- `legal.html` / `privacy.html` — starter pages
- `assets/desktop-home.png`, `tablet-home.png`, `phone-home.png` — approved visual layers
- `styles.css`, `script.js`

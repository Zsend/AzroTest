# CragLink Power — Exact Mockup Publish Prototype

This package is image-locked to the approved CragLink Power mockup so the homepage matches the visual direction exactly instead of being reinterpreted.

## Files
- `index.html` — customer-facing homepage using the approved desktop/mobile mockup artwork with functional founder-list modal.
- `design-board.html` — the full approved responsive mockup board.
- `field-team.html` — simple placeholder page for field-team stories.
- `assets/desktop-home.png` — exact approved desktop homepage crop.
- `assets/mobile-home.png` — exact approved mobile crop.
- `assets/approved-responsive-mockup.png` — full board image.

## Publish
Upload this folder to Netlify, Vercel, Cloudflare Pages, GitHub Pages, or any static host.

## Forms
The founder form uses Netlify Forms markup. On another platform, connect it to Tally, Formspree, HubSpot, Shopify, or your CRM.

## Note
This version prioritizes exact visual fidelity. A later editable/SEO-first rebuild should recreate the approved artwork as semantic HTML components once the final design is locked.

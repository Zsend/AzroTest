# CragLink Power — Exact Hybrid Publish Site

This package is built to preserve the approved visual direction while making the buttons and flows functional.

## What this version does
- Uses the approved design image as the visual source of truth.
- Uses 4x upscaled WebP assets to reduce browser pixelation.
- Adds crisp HTML buttons/navigation over the fuzzy generated button labels.
- Adds real clickable hotspots, modal, founder form validation, mobile menu, search modal, and footer email signup.
- Includes desktop, tablet, and phone artwork states.

## Publish instructions
Upload the full folder contents, including:
- index.html
- field-team.html
- legal.html
- privacy.html
- success.html
- styles.css
- script.js
- /assets

Do not upload only index.html. The site needs CSS, JavaScript, and assets to render and work.

## Form capture
The forms work visually and validate in static preview. For real lead capture, connect the form to Netlify Forms, Tally, Formspree, HubSpot, Shopify, or another backend.

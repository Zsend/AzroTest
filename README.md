# CragLink Final Color-Theory Production Site

## Files
- `index.html` — main homepage
- `index-standalone.html` — lightweight redirect to homepage
- `field-team.html`, `legal.html`, `privacy.html`, `success.html` — supporting pages
- `styles.css`, `script.js`
- `assets/` — hero, system, and expedition background art plus logo and center system image

## What changed
- Updated the middle green section so the background is **landscape only** with no product in the backdrop.
- Tuned the page around a deliberate **cool → green → warm** color progression for better harmony:
  - hero: blue/cyan aurora
  - middle section: moss/emerald field architecture
  - closing section: copper/gold expedition warmth
- Integrated the large art as true **section backgrounds** instead of stamped foreground panels.

## Publish
Upload the full folder contents to your web host root.
Open `index.html` as the homepage.

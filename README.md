# Willow & Finn Doodle Puppies Website Starter

This is a static website draft for a professional, simple, modern puppy site.

## What is included

- `index.html` — the full website content.
- `styles.css` — all styling.
- `script.js` — gallery filtering, mobile navigation, and lightbox behavior.
- `assets/photos/` — all 51 uploaded photos converted/optimized as website-safe JPGs.
- `assets/thumbs/` — smaller versions used for the gallery grid.
- `assets/videos/` — empty folder ready for future videos.

## Before publishing

1. Replace `your-email@example.com` in `index.html` with the real contact email or connect a proper form service.
2. Confirm parent names, breeds, ages, and health-test language.
3. Remove any photos that should not be public, especially photos with children, private home details, screenshots, or anything unprofessional.
4. Add actual puppy photos only after puppies are born.
5. Add deposit, pricing, and reservation policies only after you are ready to stand behind them.

## Easy edits

- Change the site title in `index.html` near the top.
- Edit parent details in the `#parents` section.
- Edit gallery captions and filters in `script.js`.
- Remove an image from the gallery by deleting that image entry from the `photos` array in `script.js`.
- Add a video by placing it in `assets/videos/` and replacing the video placeholder section in `index.html`.

## Suggested launch path

This can be hosted cheaply on Netlify, Vercel, Cloudflare Pages, GitHub Pages, or any basic web host. For forms, use Netlify Forms, Formspree, Google Forms, or a CRM form.

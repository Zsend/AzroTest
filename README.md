# Mark's Ark, Inc. — First-Version Presence Website

This is a static brand/local SEO website for Mark's Ark, Inc. It intentionally does **not** include e-commerce, checkout, live inventory, or pricing.

## Pages

- `index.html` — main landing page
- `reptiles.html` — reptile/exotic SEO page
- `aquatics.html` — freshwater fish/aquarium SEO page
- `supplies.html` — feeders, food, tanks, terrariums, supplies page
- `visit.html` — phone, hours, directions, FAQ
- `sitemap.xml` and `robots.txt` — search-engine basics

## Brand direction

Dark, old-school, exotic, energetic, and local. The site is designed to feel closer to their Instagram/new-arrivals personality than a generic clean pet-store template.

## Launch checklist

1. Replace placeholder visual slots with approved store or Instagram photos.
2. Confirm final hours, phone, address, and social links.
3. Confirm whether `https://marksarkutah.com/` is the production domain.
4. Upload the files to the hosting account or GitHub Pages.
5. Submit the sitemap in Google Search Console.
6. Make sure Google Business Profile links to the new site.

## Notes

The site uses original design elements and does not scrape or rehost Instagram images. Add real photos only after Mark's Ark approves their use.

# Future Builder AI-Control Sites — Final Static Bundle

This package contains the final static sites requested:

1. `index.html` — master hub / site map
2. `future-builder-academy/index.html` — youth comprehensive course site
3. `ai-systems-operator/index.html` — professional/operator course site
4. `founder-control-track/index.html` — private founder mastery track
5. `chew-gang-foundation-original/index.html` — preserved original attached Chew Gang course

## Deployment

No build system is required. Upload the entire folder to any static host:

- Cloudflare Pages
- Netlify Drop
- GitHub Pages
- Vercel static hosting
- Any ordinary web server

The bundle uses only static HTML, CSS, and JavaScript. There are no external dependencies.

## Suggested use

- Use `future-builder-academy/` as the public youth flagship.
- Use `ai-systems-operator/` as the adult/professional course landing site.
- Keep `founder-control-track/` private or password-protected.
- Keep `chew-gang-foundation-original/` as the preserved Level 0 foundation.

## Safety boundary

The youth course should not include unsupervised high-voltage electronics, uncontrolled agents, public posting, spending, API keys, or AI messaging strangers. The professional and private tracks should require approval gates for risky actions.

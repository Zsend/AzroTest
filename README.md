# AZRO Bitcoin Reserve Program — Final Business Hybrid Site

This build keeps the original AZRO Systems header language and dark brand framing, but shifts the page into a cleaner white business presentation.

## What changed in this version
- tightened the hero so the primary buttons land on screen sooner
- shortened and sharpened the headline for better balance on large desktops
- kept the original AZRO-style header while preserving the white business feel below it
- replaced the bulky hero stat cards with compact proof chips
- pulled the floating dashboard callouts inward so the composition stays cleaner
- preserved the one-page business funnel, reserve brief, FAQ, application flow, and legacy route redirects

## Files
- `index.html` — homepage
- `style.css` — full styling
- `script.js` — nav, FAQ, and form behavior
- `assets/` — logos, imagery, OG card, and favicons
- `pdfs/` — reserve brief, privacy, risk, and terms PDFs

## Deployment
Upload the full folder contents to your static host.

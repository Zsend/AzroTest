# Justice Grows

**Freedom. Work. Ownership.**

A responsive GitHub Pages release for a verified cannabis-justice registry, candidate-owned Mobility Passport, fair-chance employer coalition, skills-first matching, and auditable impact dashboard.

## Publish

Read `START_HERE.md`. The ZIP is repository-ready: `index.html` belongs at the root of `zsend/AzroTest`.

## Test

Open `ops.html` and enter `pilot-admin`. Use fictional information only. The default database is empty and browser-local.

## Production boundary

This Pages release is safe for public product testing, not real sensitive-data collection. Real intake requires a secure API, managed database, named accounts, MFA, encryption, logging, correction operations, and legal review. See `GO_LIVE_CHECKLIST.md`.

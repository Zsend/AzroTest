# CragLink Power — final publish-ready static site

This package is a real static website built to match the approved CragLink visual direction: dark premium shell, Arctic Cyan hero, Glacier Mint / Seafoam Sage system section, and Alpine Gold / Copper footer.

## Files

- `index.html` — homepage
- `field-team.html` — field proof / athlete seeding page
- `legal.html` — compatibility and claims note
- `privacy.html` — starter privacy page
- `success.html` — no-JS form success fallback
- `styles.css` — full responsive styling
- `script.js` — mobile menu, smooth scroll, modals, form feedback
- `assets/` — logo, product visuals, backgrounds, favicon, OG image

## What works now

- Navigation links scroll or route correctly
- Mobile menu opens and closes
- Search/cart/account icons open modal states
- Shop Systems opens the founder-list flow
- Founder forms validate email and show success state
- Forms are Netlify-compatible and also save demo submissions to localStorage during preview
- Responsive layout for desktop, tablet, and phone

## Before paid traffic

Connect the forms to your production stack: Netlify Forms, Tally, HubSpot, Shopify, Formspree, or a custom backend. The current JS prevents the site from feeling broken in static preview, but real lead capture should be connected before launch.

## Publish

Upload the folder contents to Netlify, Vercel, Cloudflare Pages, GitHub Pages, or your static host.

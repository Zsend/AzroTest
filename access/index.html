How to use Reserve Standard in 10 minutes.

This page should sit behind Cloudflare Access or another approved-user gate before live deployment.

Use BTCUSD on the weekly chart. Leave defaults unless Reserve Standard walks through changes with you.
Open TradingView and load BTCUSD on the 1W chart.
Add the Reserve Standard Weekly BTC Model from invite-only scripts.
Set the weekly reserve amount.
Set Start date, starting BTC, and starting cash only if they apply.
Read This Week. The dashboard uses the last completed week.
Review the Reserve Record and Model vs Weekly Baseline.
Record the action the same way each week.

Dashboard language

  • ADD means the model is using this week’s reserve amount.
  • WAIT means the model is preserving this week’s reserve amount in cash.
  • REDUCE is optional and only appears in managed modes. Review before acting.

Boundary

The model does not custody funds, execute orders, move BTC, or replace business judgment.

# Deployment and Send Rules

Allowed: private founder review, counsel review, advisor review, password-protected partner staging, EVIDENCE/Dan follow-up only after request.

Not allowed yet: indexed public website, public launch, medical/health/safety claims, "safe vape", "safest", "certified", "approved", active Proof Verified mark, cannabis sales, plant-touching operations, technical mechanism disclosure before IP review, competitor accusations, or sending historical source PDFs.

Recommended: keep `robots.txt`, `X-Robots-Tag`, and noindex meta tags intact and add platform-level password protection.

# Deployment and Send Rules

## Recommended use now

Keep this as a private review site. Do not publish publicly.

## When to send

Do not send this site first. The first Dan/EVIDENCE message should be no-attachment. If they ask for more after a call or clear interest, send `evidence.html` or a private hosted equivalent.

## Before public launch

Public launch requires, at minimum:

- trademark/name review;
- IP disclosure strategy;
- claims/legal review;
- PVS/CVA reviewed for external use;
- no unsupported health/safety language;
- no active Proof Verified claim;
- no public QuartzPath mechanism disclosure;
- no cannabis sales or plant-touching claims by Proof Labs.

## Technical notes

The site is static HTML/CSS/JS and can be hosted on Netlify, Vercel, Cloudflare Pages, GitHub Pages, Webflow custom code, or Framer custom code. For private review, use a password-protected host.

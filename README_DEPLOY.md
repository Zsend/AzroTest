# A Right For All — Final Initiative Website v15

This package preserves the complete public movement site, 21 original letter PDFs, 21 letter previews, nine readable excerpts, the documentary tablet image, the Free Floor standard, The Minds We Miss, and the full Unusual Thinkers Challenge.

## Production polish

- responsive AVIF/WebP/JPEG image delivery
- intrinsic image dimensions to prevent layout shift
- high-priority preload only for the hero evidence image
- optimized letter previews with first-row eager loading and remaining images lazy-loaded
- no continuous parallax or blur filters
- one IntersectionObserver reveal system
- unified spacing, grid, card, and responsive rules
- non-overlapping letter evidence presentation
- complete Challenge packet autosave/export functionality preserved
- contact route: hello.arightforall@gmail.com

A Right For All is presented as an independent public-interest initiative in development. The site does not claim nonprofit, tax-exempt, pilot, partner, vendor, facility, or government status.

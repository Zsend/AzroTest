# CragLink Power — Final customer-ready static site

## What this package includes

- `index.html` — responsive landing page
- `styles.css` — complete sky / canopy / ground color system
- `script.js` — sticky nav, mobile menu, smooth scrolling
- `assets/` — SVG logo mark, logo lockup, favicon, and product architecture illustration

## Form setup

The waitlist form is configured for Netlify Forms out of the box:

```html
<form name="founder-waitlist" method="POST" data-netlify="true" netlify-honeypot="bot-field">
```

If deploying somewhere else, connect the form to Tally, Formspree, HubSpot, Shopify, Klaviyo, ConvertKit, or your CRM of choice.

## Brand decision

The color system intentionally maps to the product story:

- Header / hero: Arctic Cyan + Midnight Navy = sky, satellite signal, precision, technical trust.
- Middle: Glacier Mint + Seafoam Sage = plants, recovery, calm, outdoor credibility.
- Footer: Alpine Gold + Copper Ember = dirt, rock, warmth, durability, field craft.

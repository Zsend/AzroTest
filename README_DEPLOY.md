# A Right For All — Final Public Site

Static production package for `arightforall.com`.

## Deploy

Upload the **contents of this directory** to the web root. The package includes a `CNAME` file for GitHub Pages, sitemap, robots file, manifest, social metadata, custom 404 page, and all public evidence assets.

No server runtime is required. The Help Build form opens a drafted email to `hello@arightforall.com`. The Unusual Thinkers Challenge stores answers locally in the visitor’s browser and exports Markdown or JSON; it does not transmit submissions.

## Final gate

The site is technically publish-ready. Before public launch, complete founder approval plus rights, privacy, provenance, and legal review for names, letters, PDFs, typed excerpts, and the founder-provided tablet image.

See `_validation/SITE_VALIDATION_REPORT.md` and `_validation/FINAL_LAUNCH_CHECKLIST.md`.

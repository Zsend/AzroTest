# CragLink Power — customer-ready prelaunch site

This is a responsive static site package for CragLink Power. It is designed as a polished prelaunch/founder-list site, not a final ecommerce storefront.

## Files

- `index.html` — main landing page
- `thank-you.html` — post-form thank-you page
- `styles.css` — complete responsive design system
- `app.js` — mobile navigation and reveal behavior
- `assets/craglink-logo.svg` — final lockup SVG
- `assets/craglink-symbol.svg` — final symbol SVG
- `assets/favicon.png` — browser icon

## Deployment

The form is Netlify-ready via `data-netlify="true"`. To use another provider, replace the form action/attributes with Tally, Formspree, HubSpot, Shopify, or your CRM endpoint.

Before sending paid traffic:

1. Confirm final product specs, runtime language, and compatibility claims with engineering.
2. Replace any placeholder language with final warranty, support, shipping, and legal copy.
3. Add privacy policy and terms pages.
4. Connect analytics and conversion tracking.
5. Test the form on the actual hosted domain.

## Design strategy

The color system is built around a vertical outdoor metaphor:

- Header/hero: Arctic Cyan + Midnight Navy = sky, satellite, trust, precision.
- Middle: Glacier Mint + Seafoam Sage = canopy, nature, endurance, calm.
- Footer/CTA: Alpine Gold + Copper Ember = sun, ground, durability, warmth.

The logo top treatment is intentionally clean: white sparkle/dots with a single controlled mast fade into cyan. No random multi-color dots or broken color segments.

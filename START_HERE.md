# Start here

This is the final Justice Grows GitHub Pages test release for:

`https://zsend.github.io/AzroTest/`

## Publish in five steps

1. Back up the current `zsend/AzroTest` repository.
2. Copy **everything in this folder** into the repository root, preserving `.git/`.
3. Commit and push to the default branch.
4. In GitHub, choose **Settings → Pages → Build and deployment → GitHub Actions**.
5. Wait for **Deploy Justice Grows to GitHub Pages** to pass, then hard-refresh the site.

Detailed commands are in `DEPLOY_TO_AZROTEST.md`.

## What zero means

A fresh deployment intentionally shows zero verified employers, roles, starts, and outcomes. Zero is not a broken empty state. It means no record has completed the relevant proof path in the active data plane.

## Before real submissions

Do not invite real candidate, custody, legal-document, employer-contact, correction, or release submissions until the secure remote data plane and every blocking item in `GO_LIVE_CHECKLIST.md` are complete.

# A Right For All — v16 Deployment Guide

## Recommended contact email

Create and test:

`hello.arightforall@gmail.com`

The site is already configured to use this address.

## GitHub Pages test deployment

Target:

`https://zsend.github.io/AzroTest/`

1. Extract `A_Right_For_All_AzroTest_Final_v16.zip`.
2. Upload the **contents inside the ZIP** directly to the root of the `zsend/AzroTest` Pages branch.
3. Commit and wait for GitHub Pages to rebuild.
4. Hard-refresh the test URL.
5. Test the contact form, mobile menu, letter PDFs, Challenge autosave/export, and all main navigation links.

The test build blocks search indexing.

## Future production deployment

1. Use `A_Right_For_All_Public_Initiative_Final_v16.zip`.
2. Complete founder approval and the narrow privacy/provenance review for public evidence.
3. Configure the custom domain deliberately in GitHub Pages or the selected host.
4. Add the host-generated `CNAME` only when you are ready to connect `arightforall.com`.
5. Confirm HTTPS and then submit the production sitemap.

The production ZIP does not include a `CNAME` and will not connect the domain automatically.

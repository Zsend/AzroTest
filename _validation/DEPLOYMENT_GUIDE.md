## Public contact

## Upload

Upload the **contents of this folder**, not the enclosing folder, to the hosting root.

### Test build

Target: `https://zsend.github.io/AzroTest/`  
Search indexing is blocked during testing.

### Production build

Target: future `https://arightforall.com/`  
No `CNAME` is included; domain connection remains a separate deliberate step.

## After upload

1. Wait for the host to finish publishing.
2. Hard-refresh the browser.
3. Test Home, Letters, Free Floor, Bridge, Minds We Miss, Challenge, Evidence, About & Privacy.
4. Send a test message to `hello.arightforall@gmail.com`.
5. Export one Challenge packet in Markdown and JSON.
6. Open the safety-redacted letter preview and PDF to confirm the public version only.

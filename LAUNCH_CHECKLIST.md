# Launch checklist — test-ready GitHub Pages deploy

- [ ] Unzip the package locally
- [ ] Upload the **contents** to the repository root
- [ ] Confirm the repo is public if using GitHub Free
- [ ] Open **Settings → Pages**
- [ ] Set **Source** to **Deploy from a branch**
- [ ] Set **Branch** to `main` and **Folder** to `/ (root)`
- [ ] Wait for the Pages URL to appear
- [ ] Open `index.html`, `letters/index.html`, `unusual-thinkers/index.html`, and one letter excerpt after deploy
- [ ] Confirm the join email opens correctly
- [ ] Confirm the original PDFs open correctly
- [ ] Confirm you are comfortable with the tablet photo and letters being public
- [ ] Set up `hello@arightforall.com` or forwarding before wider launch

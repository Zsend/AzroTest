# Launch checklist

- [ ] `hello@arightforall.com` exists or forwards to a real inbox
- [ ] You uploaded the **unzipped contents** to the repo root
- [ ] GitHub Pages is set to deploy from `main` → `/ (root)`
- [ ] The homepage loads at the GitHub Pages test URL
- [ ] The letters archive opens and the original PDFs download
- [ ] The Sources page loads and external links open
- [ ] The join form opens a drafted email correctly
- [ ] Social preview image renders correctly when you share the site
- [ ] Only after testing: connect `arightforall.com` and add the live `CNAME` file

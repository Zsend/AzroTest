# Publish on GitHub Pages

1. Copy all files in this folder into the root of the production repository.
2. Commit and push to `main`.
3. In GitHub: Settings → Pages → Deploy from a branch → `main` / root.
4. Set the custom domain to `prohibitioncareers.com`.
5. In GoDaddy, preserve all Purelymail MX/TXT/CNAME records. Add only the GitHub Pages web records GitHub displays.
6. Wait for DNS and certificate provisioning, then enable **Enforce HTTPS**.
7. Test every navigation link and all three private intake flows on phone and desktop.

Do not upload browser-saved pages, tokens, exports, or files named `saved_resource*`.

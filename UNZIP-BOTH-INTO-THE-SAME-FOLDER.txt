AZRO site package setup

Do this exactly:

1) Create a new empty folder on your computer.
2) Unzip 1-azro-site-unzip-first.zip into that empty folder.
3) Unzip 2-azro-xrp-videos-unzip-second.zip into that same exact folder.
4) Publish the resulting files and folders with index.html at the root.

Important:
- Do NOT drag home-gallery, products-gallery, or videos on top of each other.
- This build was changed so the 6 fixed XRP videos now live in one dedicated folder:
  xrp-videos/
- The second zip only adds that one folder, so there should be no replace prompts.

Expected folder after both zips are unzipped:
- index.html
- features.html
- products.html
- xrp-videos/
- home-gallery/
- products-gallery/
- videos/
- downloads/
- pdfs/

The 6 required files inside xrp-videos/ are:
- xrp-home-dashboard-full.mp4
- xrp-home-dashboard-simple.mp4
- xrp-product-workflow-full.mp4
- xrp-product-workflow-action.mp4
- xrp-dashboard-full.mp4
- xrp-dashboard-simple.mp4

Why this package exists:
The earlier split package used the same folder names as the site, which caused Finder replace prompts. This package avoids that by using one unique folder for the fixed XRP videos.

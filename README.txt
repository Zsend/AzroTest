AZRO Systems BTC Operating System — complete HTML site bundle

Files
- index.html .............. full landing page
- style.css .............. extracted AZRO site design system
- btc-os.css ............. BTC Operating System custom styling
- script.js .............. navigation / sticky CTA behavior
- assets/screens/ ........ dashboard screenshots used in the page
- exports/ ............... screenshot-ready PNG gallery images
- gallery.html ........... quick preview page for the PNG gallery

How to use
1. Open index.html in a browser.
2. Open gallery.html if you just want the screenshot-ready PNG set.
3. Replace any CTA href values once the final Gumroad product URL is ready.

Notes
- The page references Exo 2 from Google Fonts. If you open the files offline, the page will fall back to system fonts until network access is available.
- The included PNGs are the screenshot-ready gallery images from the current design direction.
- Support email used in the page: support@azrosystems.com

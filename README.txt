AZRO Systems BTC Operating System — publish-ready kit

Use this folder in one of two ways:

1) Direct upload:
   Upload the PNG files from /exports to Gumroad.

2) Manual screenshot:
   Open capture-gallery.html, click any "Open slide" button, then screenshot the full browser window.
   Each slide page contains only one 16:9 artboard and is scaled to fit the viewport.

Files:
- capture-gallery.html  -> main review / link hub
- full-site.html        -> stacked preview page
- slide-01-cover.html   -> single slide page
- slide-02-process.html -> single slide page
- slide-03-modes.html   -> single slide page
- slide-04-accountability.html
- slide-05-why-policy.html
- slide-06-fit.html
- slides/*.svg          -> crisp vector originals
- exports/*.png         -> 3840x2160 upload-ready PNGs

AZRO Systems BTC OS — complete screenshot site bundle

What is included
- 6 fixed 16:9 HTML slide pages
- a capture gallery with direct links to every slide
- a stacked review page
- local assets folder with logo, background, and favicon

How to use
1. Open capture-gallery.html
2. Click the slide you want
3. Screenshot the full browser window
4. Upload those screenshots to Gumroad

Notes
- This bundle is self-contained and should work offline
- Each slide is a fixed 16:9 artboard
- No export PNG dependencies are required

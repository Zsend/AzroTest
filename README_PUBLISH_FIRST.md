# Reserve Standard — GitHub Pages publish pack

This folder is ready to publish to a GitHub Pages repository or subdirectory.

## For the test URL
Target:
`https://zsend.github.io/AzroTest/`

## Publish steps
1. Open your `AzroTest` GitHub Pages repo or folder.
2. Delete the old site files in the publish root.
3. Upload the **contents of this folder** directly into the repo root for that site.
4. Make sure `.nojekyll` stays in the root.
5. Wait for GitHub Pages to refresh.

## Important
- The site is static HTML/CSS/JS. No build step is required.
- The review flow uses `review@reservestandard.com` and `treasury@reservestandard.com`.
- The PDFs are already included under `assets/docs/`.

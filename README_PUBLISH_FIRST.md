# Reserve Standard — Publish-Ready Site

This folder is ready to publish as a static GitHub Pages project site.

## For the test URL
Publish these files at the repository root for:
`https://zsend.github.io/AzroTest/`

## Publish steps
1. Create or open the GitHub repository used for the test site.
2. Upload **everything in this folder** to the repository root.
3. Make sure `.nojekyll` stays in the root.
4. In GitHub, enable Pages for the repository and publish from the selected branch / root.
5. Wait for the Pages deploy to finish.

## Before public launch
- create `treasury@reservestandard.com`
- create `review@reservestandard.com`
- update `CNAME.example` to `CNAME` only when the final custom domain is ready
- update canonical / sitemap URLs once the production domain replaces the test URL

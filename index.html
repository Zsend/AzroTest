One weekly model. One clean reserve action.

The model is designed for BTCUSD on the weekly chart. It turns a chosen reserve amount into a reviewable weekly action without making the owner live inside charts.

Weekly Baseline

The model path is compared against a simple weekly baseline using the same reserve amount.

The proof surface is simple: same reserve amount, better BTC outcome.

Reserve Standard does not need hype. The question is whether the same reserve budget can produce more BTC, more sats, and a clearer record over time.

Operating cash, payroll cash, tax cash, vendor cash, and known near-term obligations stay outside the model.

Why this is a real leverage point

This is not a moonshot hardware problem.

The infrastructure is already deployed. The time is already being taken. The first job is to make the free side of the device worthy of a human being.

  • If a device can already carry communication and paid entertainment, it can carry a humane minimum.
  • If the system can already occupy attention, it can stop wasting it.
  • If a person is going to wait there anyway, waiting should not mean mental decay by default.

Inside the room

Start where the movement actually starts: inside the room.

This is not a policy brochure problem. It is a human time problem. A person can wake up in the same room, under the same artificial light, with the same pressure in their chest, the same uncertainty about court or transfer, and the same device sitting nearby. The device is real. The growth path still may not be.

When a person is locked in a room for days, months, or years without sunlight, fresh air, or even a book, a tablet should not be better at extracting money than helping them build a future.

The contradiction

The problem is simple enough to explain in one sentence.

The system already proved that a correctional tablet can be distributed at scale. It still has not guaranteed that the free side of that device will actually help a person learn, stabilize, or prepare for release.

What the system already proved

The rail exists.

  • Devices can already reach people in confinement.
  • Those devices already carry paid communication and entertainment.
  • They already shape daily attention inside.
  • The technical question is no longer whether a screen can be in the room.

What still may not be guaranteed

A real free floor.

  • Free full GED prep on the device itself.
  • Basic daily wellness tools for people in survival mode.
  • Progress that survives movement through jail, prison, and release.
  • A free section built to improve a life, not merely satisfy the bare minimum.

This is the simplest profound fix sitting in plain sight: if the device is already there, the free section should teach, stabilize, and open a path forward. It should not stop at the point where a person needs it most.

The standard

A real movement needs a real minimum.

A Right For All is not starting with a vague promise to fix everything. It is starting with a humane digital floor that should exist on every correctional tablet where the hardware is already deployed.

1

Free full GED prep

A serious, usable path to the GED should not depend on whether a person can pay inside. The free side of the device should teach.

2

Basic daily wellness

People in confinement are often operating under trauma, stress, and survival mode. The free floor should include practical tools to help them regulate and think clearly.

3

Continuity through the system

If someone starts learning in county jail, that progress should not vanish when they move to prison or return to the community.

The first standard is not the whole build

The larger vision is a daily workbook built for confinement, recovery, and reentry.

The minimum gets the movement off the ground. The long-term product vision goes further: a structured daily system that helps a person stabilize, learn, set goals, and carry progress forward.

What the fuller build can grow into

  • A daily check-in that responds to where a person actually is.
  • Trauma-aware routines that help someone calm down enough to learn.
  • A guided path through education, habit building, and preparation for release.
  • Progress that follows the person across county jail, prison, and life after release.

How it moves

This can move because the constraint is not magic. It is alignment.

The site is designed to attract serious people because serious people need more than outrage. They need a path. The first path is disciplined: document the reality, set the standard, activate existing rails, and make the baseline durable.

1

Document the real gap

Collect founder testimony, letters from inside, and public record so the case is vivid without becoming sloppy.

2

Set the humane minimum

Make the first demand simple enough to rally around and specific enough to build toward: free GED prep and basic wellness on the device already in the room.

3

Activate existing rails

Use the hardware, content, partners, and system openings that already exist instead of pretending this has to start from zero.

4

Turn proof into permanence

Once the free floor is real in practice, press it into policy, procurement, and broader adoption.

Why this is a leverage point

The same fix can change the emotional story and the operating story at once.

It matters morally because people are wasting away in rooms where the device is already present. It matters operationally because a stronger free floor can be piloted, improved, and scaled without waiting for a whole new physical system.

  • It starts where human time is currently being wasted.
  • It uses a distribution rail that already exists.
  • It creates an immediate wedge for educators, operators, builders, and funders to work together.
  • It opens the path to the larger workbook vision instead of replacing it.
